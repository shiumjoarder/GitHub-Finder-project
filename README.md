# GitHub Finder 🔍

A web app to search GitHub users and view their profile and repositories.

## Features

- Search any GitHub user by username
- View profile info (avatar, name, bio, followers, following)
- View all public repositories with links
- Error handling for invalid usernames
- Clean dark UI

## Tech Used

- HTML
- CSS
- JavaScript (Fetch API, Async/Await)
- GitHub REST API

## How to Use

1. Type a GitHub username in the search box
2. Click Search
3. View profile and repositories
